# 사용자 노출 문구 수정 내역

## 기준 문서

- `전기프 1조_2차 기획서 최종수정판 (1).pdf`의 6장 프롬프트
- `전기프 1조 설계 문서 원판 (1).pdf`의 고객 안내 문구 및 표시 규칙

## 수정 원칙

- 메뉴명, 입력 안내, 오류·완료 메시지, 목록 표시 형식만 수정했다.
- 예약 가능 조건, 상태 전이, 파일 저장 조건 등 비즈니스 로직은 수정하지 않았다.
- 로직 변경이나 문서 간 기준 결정이 필요한 항목은 `로직_차이_검토.md`에 별도로 정리했다.

## 반영 내용

### 초기 화면 및 회원가입

- 초기 화면에 현재 기준 시간을 표시하도록 수정했다.
- 초기 메뉴의 잘못된 입력 안내를 `메뉴 번호는 1 또는 2를 입력해야 합니다.`로 수정했다.
- 프로그램 종료 문구, 비밀번호 조건 안내, 역할 입력 안내와 오류 문구를 기획서 표현에 맞췄다.
- 기획서에 없는 회원가입 완료 문구를 제거했다.

### 고객 메뉴 및 예약

- 고객 메뉴명과 업소·방·숙박 인원 선택 안내를 기획서 표현에 맞췄다.
- 방 목록의 체크인·체크아웃 시간 구분자를 기획서 형식에 맞췄다.
- 예약 날짜, 체크인 시간, 체크아웃 시간 입력 안내와 오류 문구를 정리했다.
- 예약 완료 문구와 예약 정보 요약 출력 순서를 기획서 흐름에 맞췄다.
- 영업정지 기간과 겹치는 예약의 안내 문구를 기획서 표현으로 수정했다.

### 고객 기록조회

- 예약 목록에 상태 코드, 인원, 생성 시각, 평점 작성 여부가 표시되도록 형식을 수정했다.
- 기획서에 없는 `기록조회 메뉴` 제목을 제거했다.
- 입실·퇴실·평점 작성의 입력 안내와 완료 문구를 기획서 표현에 맞췄다.
- 예약 변경 완료 안내와 취소 대상 선택 안내를 기획서 표현에 맞췄다.
- `패널티` 표기를 문서에서 사용하는 `페널티`로 통일했다.

### 사장 및 관리자 메뉴

- 사장 메뉴명과 `업소 정보 수정` 항목명을 기획서 표현에 맞췄다.
- 내 방 목록의 표시 순서를 `방 번호, 인원 수, 체크인 시간, 체크아웃 시간`으로 수정했다.
- 업소·방 수정 선택 안내와 중복 우편번호 안내를 정리했다.
- 관리자 메뉴명, 시간 관리 항목명, 새 기준 시간 입력 안내를 기획서 표현에 맞췄다.
